
# 📡 نکسا

ساخت خودکار پلی‌لیست IPTV فارسی با دریافت داده از مخزن iptv-org و تولید فایل M3U به‌صورت اتوماتیک.

---

## 🚀 درباره پروژه

این پروژه به‌صورت خودکار:

1. دریافت لیست کانال‌ها از مخزن رسمی iptv-org
2. فیلتر کردن کانال‌ها بر اساس زبان (پیش‌فرض: فارسی)
3. دریافت لیست استریم‌ها
4. ادغام اطلاعات کانال و استریم
5. تولید سه فایل نهایی IPTV با فرمت `.m3u`
6. بروزرسانی روزانه از طریق GitHub Actions
7. جداسازی کیفیت SD و HD

---

## ⚙ نحوه کار

پروژه از دو منبع رسمی استفاده می‌کند:

- Channels Database (CSV)
- Streams API (JSON)

پس از فیلتر زبان، اطلاعات با یکدیگر Merge شده و خروجی نهایی به شکل یک فایل آماده استفاده IPTV تولید می‌شود.

---

## 📂 خروجی نهایی

سه فایل فایل نهایی:

```

playlist.m3u
playlist_sd.m3u
playlist_hd.m3u

````

قابل استفاده در:

- VLC
- IPTV Smarters
- Kodi
- Smart TV
- هر پلیر سازگار با M3U

---

## 🔄 بروزرسانی خودکار

این پروژه از GitHub Actions استفاده می‌کند و:

- هر روز به‌صورت خودکار اجرا می‌شود
- هنگام Push به شاخه main اجرا می‌شود
- امکان اجرای دستی نیز دارد

---

## 🛠 اجرای دستی

اگر بخواهید به‌صورت لوکال اجرا کنید:

```bash
pip install pandas requests
python iptv_builder.py
````

---

## 🧠 تنظیم زبان

در فایل `iptv_builder.py` می‌توانید زبان موردنظر را تغییر دهید:

```python
LANGUAGE_FILTER = "fas"
```

مثال:

* `eng` برای انگلیسی
* `ara` برای عربی
